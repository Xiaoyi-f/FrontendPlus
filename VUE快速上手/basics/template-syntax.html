Vue模板语法

Vue.js使用基于HTML的模板语法，允许开发者声明式地将DOM绑定到底层Vue实例的数据。所有Vue.js模板都是合法的HTML，所以能被遵循规范的浏览器和HTML解析器解析。

模板语法核心概念

  • 插值：使用双大括号{{}}进行文本插值
  • 指令：带有v-前缀的特殊属性，用于执行DOM操作
  • 事件处理：使用v-on指令处理DOM事件
  • 条件渲染：根据条件展示或隐藏元素
  • 列表渲染：基于数据数组渲染列表项
  • 表单输入绑定：使用v-model实现双向数据绑定

文本插值

文本插值是最基本的模板语法，使用双大括号{{}}进行数据绑定。插值内容会被解析为文本内容。

文本插值示例
<div id="app">
  <h1>{{ title }}</h1>
  <p>欢迎来到{{ framework }}的世界！</p>
  <p>当前计数: {{ count }}</p>
</div>

<script>
const { createApp } = Vue;

createApp({
  data() {
    return {
      title: 'Vue模板语法',
      framework: 'Vue.js',
      count: 0
    }
  }
}).mount('#app');
</script>

使用JavaScript表达式

在插值中可以使用JavaScript表达式，但只能是单个表达式，不能是语句。

JavaScript表达式示例
<div id="app">
  <p>{{ message.split('').reverse().join('') }}</p>
  <p>{{ count > 5 ? '大于5' : '小于等于5' }}</p>
  <p>{{ user.firstName + ' ' + user.lastName }}</p>
</div>

<script>
const { createApp } = Vue;

createApp({
  data() {
    return {
      message: 'Hello Vue',
      count: 8,
      user: {
        firstName: '张',
        lastName: '三'
      }
    }
  }
}).mount('#app');
</script>

指令

指令是带有v-前缀的特殊属性，其职责是当表达式的值改变时响应式地作用于DOM。

指令 描述 示例
v-text 更新元素的文本内容 v-text="message"
v-html 更新元素的innerHTML v-html="rawHtml"
v-show 根据条件显示/隐藏元素 v-show="isVisible"
v-if 条件性渲染元素 v-if="condition"
v-else v-if的else分支 v-else
v-else-if v-if的else-if分支 v-else-if="condition2"
v-for 基于数组/对象渲染列表 v-for="item in items"
v-on 绑定事件监听器 v-on:click="handleClick"
v-bind 动态绑定HTML属性 v-bind:href="url"
v-model 创建双向数据绑定 v-model="value"

v-bind指令

v-bind用于动态绑定HTML属性，可以缩写为:。

v-bind示例
<div id="app">
  <a v-bind:href="url">访问网站</a>
  <img v-bind:src="imageSrc" v-bind:alt="imageAlt">
  <div v-bind:class="{ active: isActive, 'text-danger': hasError }">动态类名</div>
  <div v-bind:style="{ color: textColor, fontSize: fontSize + 'px' }">动态样式</div>
</div>

<script>
const { createApp } = Vue;

createApp({
  data() {
    return {
      url: 'https://vuejs.org',
      imageSrc: '/path/to/image.jpg',
      imageAlt: '示例图片',
      isActive: true,
      hasError: false,
      textColor: 'red',
      fontSize: 16
    }
  }
}).mount('#app');
</script>

v-on指令

v-on用于监听DOM事件，可以缩写为@。

v-on示例
<div id="app">
  <button v-on:click="increment">计数 +1</button>
  <button @click="decrement">计数 -1</button>
  <button @click="reset">重置</button>
  <p>计数: {{ count }}</p>
  <input @keyup.enter="submit" placeholder="按回车提交">
</div>

<script>
const { createApp } = Vue;

createApp({
  data() {
    return {
      count: 0
    }
  },
  methods: {
    increment() {
      this.count++;
    },
    decrement() {
      this.count--;
    },
    reset() {
      this.count = 0;
    },
    submit() {
      alert('提交数据: ' + this.count);
    }
  }
}).mount('#app');
</script>

修饰符

修饰符是以半角句号.指明的特殊后缀，用于指出一个指令应该以特殊方式绑定。

事件修饰符
<!-- 阻止冒泡 -->
<button @click.stop="doThis"></button>

<!-- 阻止默认行为 -->
<form @submit.prevent="onSubmit"></form>

<!-- 修饰符可以串联 -->
<button @click.stop.prevent="doThat"></button>

<!-- 添加事件监听器时使用 capture 模式 -->
<div @click.capture="doThis"></div>

<!-- 只触发一次 -->
<button @click.once="doThis"></button>

<!-- 滚动事件的默认行为 (即滚动行为) 将会立即触发 -->
<div @scroll.passive="onScroll"></div>
按键修饰符
<!-- 按回车键触发 -->
<input @keyup.enter="submit">

<!-- 按空格键触发 -->
<input @keyup.space="handleSpace">

<!-- 按删除键触发 (包括退格键和删除键) -->
<input @keyup.delete="handleDelete">

<!-- 按下Ctrl + Enter触发 -->
<input @keydown.ctrl.enter="submit">

<!-- 按下Alt + C触发 -->
<input @keydown.alt.c="clear">

<!-- 按下Shift + PageUp触发 -->
<input @keydown.shift.page-up="onPageUp">

条件渲染

Vue提供了v-if、v-else、v-else-if和v-show指令来条件性地渲染内容。

条件渲染示例
<div id="app">
  <div v-if="type === 'A'">A</div>
  <div v-else-if="type === 'B'">B</div>
  <div v-else-if="type === 'C'">C</div>
  <div v-else>Not A/B/C</div>

  <div v-show="isVisible">v-show控制的元素</div>
</div>

<script>
const { createApp } = Vue;

createApp({
  data() {
    return {
      type: 'A',
      isVisible: true
    }
  }
}).mount('#app');
</script>

v-if vs v-show 的区别

  • v-if：条件性渲染，如果条件为假，则不会渲染元素，切换时会销毁和重建元素
  • v-show：始终渲染并保留在DOM中，只是切换CSS的display属性
  • 使用场景：v-if适合条件很少改变的场景，v-show适合频繁切换的场景

列表渲染

使用v-for指令可以基于数组/对象来渲染列表。

列表渲染示例
<div id="app">
  <ul>
    <li v-for="(item, index) in items" :key="item.id">
      {{ index }} - {{ item.name }} ({{ item.age }}岁)
    </li>
  </ul>

  <div v-for="(value, key, index) in object" :key="key">
    {{ index }}. {{ key }}: {{ value }}
  </div>

  <button @click="addItem">添加项目</button>
</div>

<script>
const { createApp } = Vue;

createApp({
  data() {
    return {
      items: [
        { id: 1, name: '张三', age: 25 },
        { id: 2, name: '李四', age: 30 },
        { id: 3, name: '王五', age: 28 }
      ],
      object: {
        name: 'Vue',
        version: '3.0',
        author: '尤雨溪'
      }
    }
  },
  methods: {
    addItem() {
      const id = this.items.length + 1;
      this.items.push({
        id: id,
        name: '新用户' + id,
        age: 20 + (id % 10)
      });
    }
  }
}).mount('#app');
</script>
