Vue实例

Vue实例是Vue.js框架的核心，每个Vue应用都是通过创建Vue实例来启动的。Vue实例负责管理一个组件的整个生命周期，包括数据响应、模板编译、DOM更新等。

Vue实例概览

  • 创建实例：通过createApp()方法创建应用实例
  • 数据响应：实例中的数据具有响应性
  • 生命周期钩子：实例在不同阶段触发的回调函数
  • 选项配置：配置实例的行为和功能
  • 方法定义：在实例中定义可复用的方法
  • 计算属性：基于响应式依赖进行计算的属性

创建Vue应用实例

在Vue 3中，我们使用createApp()方法创建应用实例，这与Vue 2中的new Vue()有所不同。

// 导入Vue
const { createApp } = Vue;

// 创建Vue应用实例
const app = createApp({
  // 组件选项
  data() {
    return {
      message: 'Hello Vue!',
      count: 0
    }
  },

  methods: {
    increment() {
      this.count++;
    }
  }
});

// 挂载到DOM元素
app.mount('#app');
方法 描述 示例
createApp(options) 创建应用实例 createApp({ data() { ... } })
app.mount(element) 将应用实例挂载到DOM元素 app.mount('#app')
app.unmount() 卸载应用实例 app.unmount()
app.component(name, component) 注册全局组件 app.component('MyButton', MyButton)

Vue实例的选项

Vue实例可以接受一个选项对象，用于配置实例的行为。以下是最常用的选项：

const app = createApp({
  // 数据选项
  data() {
    return {
      title: 'Vue实例示例',
      description: '学习Vue实例的配置选项',
      items: ['Vue', 'React', 'Angular']
    }
  },

  // 计算属性
  computed: {
    itemCount() {
      return this.items.length;
    }
  },

  // 方法
  methods: {
    addItem(item) {
      this.items.push(item);
    },

    removeItem(index) {
      this.items.splice(index, 1);
    }
  },

  // 侦听器
  watch: {
    count(newVal, oldVal) {
      console.log(`计数从 ${oldVal} 变为 ${newVal}`);
    }
  },

  // 生命周期钩子
  created() {
    console.log('Vue实例已创建');
  },

  mounted() {
    console.log('Vue实例已挂载到DOM');
  }
});

Vue 3组合式API

Vue 3引入了组合式API（Composition API），提供了一种更灵活的方式来组织组件逻辑。

选项式API vs 组合式API
// 选项式API (Options API)
createApp({
  data() {
    return {
      count: 0,
      name: 'Vue'
    }
  },
  computed: {
    doubledCount() {
      return this.count * 2;
    }
  },
  methods: {
    increment() {
      this.count++;
    }
  }
});
组合式API (Composition API)
createApp({
  setup() {
    const { ref, computed } = Vue;

    const count = ref(0);
    const name = ref('Vue');

    const doubledCount = computed(() => count.value * 2);

    const increment = () => {
      count.value++;
    };

    return {
      count,
      name,
      doubledCount,
      increment
    };
  }
});

生命周期钩子函数

Vue实例在创建到销毁的过程中会经历多个阶段，每个阶段都会触发相应的生命周期钩子函数。

钩子函数 描述 使用场景
beforeCreate 实例初始化之后，数据观测和事件配置之前 访问原始数据，不推荐使用
created 实例创建完成，数据观测、属性和方法已完成 异步数据获取、定时器设置
beforeMount 挂载开始之前被调用 模板编译完成，但尚未挂载到DOM
mounted 实例挂载到DOM后调用 访问DOM元素、发送API请求
beforeUpdate 数据更新时调用，发生在DOM更新之前 访问更新前的DOM状态
updated DOM更新完成后调用 执行依赖DOM的操作
beforeUnmount 实例卸载前调用 清理定时器、取消订阅
unmounted 实例卸载后调用 执行清理操作
生命周期钩子示例
createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  },

  beforeCreate() {
    console.log('beforeCreate: 实例初始化');
    console.log('数据未初始化: ', this.message); // undefined
  },

  created() {
    console.log('created: 数据已初始化');
    console.log('数据已初始化: ', this.message); // 'Hello Vue!'
  },

  beforeMount() {
    console.log('beforeMount: 模板编译完成');
  },

  mounted() {
    console.log('mounted: 实例已挂载到DOM');
    console.log('DOM已更新');
  },

  beforeUpdate() {
    console.log('beforeUpdate: 数据更新前');
  },

  updated() {
    console.log('updated: DOM已更新');
  },

  beforeUnmount() {
    console.log('beforeUnmount: 实例即将卸载');
  },

  unmounted() {
    console.log('unmounted: 实例已卸载');
  },

  methods: {
    updateMessage() {
      this.message = '更新后的消息';
    }
  }
}).mount('#app');

响应式数据系统

Vue的响应式系统是其核心特性之一，它使得数据变化能自动触发视图更新。

const { createApp, ref, reactive } = Vue;

createApp({
  setup() {
    // 使用ref创建响应式数据
    const count = ref(0);

    // 使用reactive创建响应式对象
    const user = reactive({
      name: '张三',
      age: 25,
      email: 'zhangsan@example.com'
    });

    const increment = () => {
      count.value++;
    };

    const updateAge = (newAge) => {
      user.age = newAge;
    };

    return {
      count,
      user,
      increment,
      updateAge
    };
  }
}).mount('#app');
