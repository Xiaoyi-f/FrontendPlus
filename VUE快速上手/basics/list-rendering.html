列表渲染

列表渲染是Vue.js的重要功能之一，它允许我们基于数组或对象来渲染列表。Vue提供了v-for指令来实现列表渲染，可以遍历数组、对象、数字等可迭代的数据结构。

列表渲染核心概念

  • v-for指令：用于基于源数据多次渲染元素或模板块
  • 数组遍历：遍历数组元素并渲染列表项
  • 对象遍历：遍历对象的属性并渲染列表项
  • 数字遍历：遍历数字范围并渲染列表项
  • key属性：为列表项提供唯一标识，提高渲染性能
  • 数组方法：使用数组变异方法来更新列表

v-for指令基础用法

v-for指令基于一个数组来渲染一个列表。v-for指令需要使用item in items形式的特殊语法，其中items是源数据数组，item是数组元素的别名。

v-for遍历数组示例
<div id="app">
  <ul>
    <li v-for="(item, index) in items" :key="item.id">
      {{ index + 1 }}. {{ item.name }} - {{ item.description }}
    </li>
  </ul>
  <button @click="addItem">添加项目</button>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
  setup() {
    const items = ref([
      { id: 1, name: 'Vue.js', description: '渐进式JavaScript框架' },
      { id: 2, name: 'React', description: '用于构建用户界面的JavaScript库' },
      { id: 3, name: 'Angular', description: '前端开发平台' }
    ]);

    const addItem = () => {
      const newId = items.value.length + 1;
      items.value.push({
        id: newId,
        name: '框架 ' + newId,
        description: '描述 ' + newId
      });
    };

    return {
      items,
      addItem
    };
  }
}).mount('#app');
</script>

遍历对象

我们也可以使用v-for来遍历对象的属性。

v-for遍历对象示例
<div id="app">
  <ul>
    <li v-for="(value, key, index) in user" :key="key">
      {{ index + 1 }}. {{ key }}: {{ value }}
    </li>
  </ul>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
  setup() {
    const user = ref({
      name: '张三',
      age: 25,
      email: 'zhangsan@example.com',
      job: '前端工程师'
    });

    return {
      user
    };
  }
}).mount('#app');
</script>

遍历数字

当v-for的表达式是数字时，会循环指定次数。

v-for遍历数字示例
<div id="app">
  <span v-for="n in count" :key="n">{{ n }} </span>
  <div>当前计数: {{ count }}</div>
  <button @click="increment">增加</button>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
  setup() {
    const count = ref(5);

    const increment = () => {
      count.value++;
    };

    return {
      count,
      increment
    };
  }
}).mount('#app');
</script>

key属性的重要性

Vue默认使用"就地更新"策略，当列表数据改变时，Vue会尽可能高效地更新DOM元素。但有时我们需要更精确地控制列表项的更新方式，这时就需要使用key属性。

情况 说明 建议
列表顺序可能改变 没有key时，Vue会就地复用元素 使用唯一key值
列表顺序固定 可以使用index作为key（不推荐） 仍建议使用唯一key值
列表项有唯一标识 使用唯一标识作为key 最佳实践
key属性的重要性示例
<div id="app">
  <h3>没有key的情况</h3>
  <div v-for="item in itemsWithoutKey">
    <input type="checkbox" :id="'nokey_' + item.id">
    <label :for="'nokey_' + item.id">{{ item.name }}</label>
  </div>
  <button @click="reverseWithoutKey">反转顺序(无key)</button>

  <h3>有key的情况</h3>
  <div v-for="item in itemsWithKey" :key="item.id">
    <input type="checkbox" :id="'withkey_' + item.id">
    <label :for="'withkey_' + item.id">{{ item.name }}</label>
  </div>
  <button @click="reverseWithKey">反转顺序(有key)</button>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
  setup() {
    const itemsWithoutKey = ref([
      { id: 1, name: '项目A' },
      { id: 2, name: '项目B' },
      { id: 3, name: '项目C' }
    ]);

    const itemsWithKey = ref([
      { id: 1, name: '项目A' },
      { id: 2, name: '项目B' },
      { id: 3, name: '项目C' }
    ]);

    const reverseWithoutKey = () => {
      itemsWithoutKey.value.reverse();
    };

    const reverseWithKey = () => {
      itemsWithKey.value.reverse();
    };

    return {
      itemsWithoutKey,
      itemsWithKey,
      reverseWithoutKey,
      reverseWithKey
    };
  }
}).mount('#app');
</script>

数组变异方法

Vue能够检测到数组变化并更新DOM，这包括以下变异方法：

数组变异方法

  • push()：在数组末尾添加一个或多个元素
  • pop()：移除数组最后一个元素
  • shift()：移除数组第一个元素
  • unshift()：在数组开头添加一个或多个元素
  • splice()：在任意位置添加/移除元素
  • sort()：对数组元素进行排序
  • reverse()：反转数组元素顺序

替换数组方法

以下方法不会改变原数组，而是返回一个新数组：

  • filter()：过滤数组元素
  • concat()：连接数组
  • slice()：截取数组
数组变异方法示例
<div id="app">
  <div>
    <input v-model="newItem" placeholder="输入新项目">
    <button @click="addItem">添加</button>
  </div>

  <ul>
    <li v-for="(item, index) in items" :key="item.id">
      {{ item.name }}
      <button @click="removeItem(index)">删除</button>
    </li>
  </ul>

  <div>
    <button @click="sortItems">排序</button>
    <button @click="reverseItems">反转</button>
    <button @click="filterItems">过滤</button>
    <button @click="clearItems">清空</button>
  </div>

  <div>总数: {{ items.length }}</div>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
  setup() {
    const items = ref([
      { id: 1, name: 'Vue.js' },
      { id: 2, name: 'React' },
      { id: 3, name: 'Angular' }
    ]);
    const newItem = ref('');
    let nextId = 4;

    const addItem = () => {
      if (newItem.value.trim() !== '') {
        items.value.push({
          id: nextId++,
          name: newItem.value
        });
        newItem.value = '';
      }
    };

    const removeItem = (index) => {
      items.value.splice(index, 1);
    };

    const sortItems = () => {
      items.value.sort((a, b) => a.name.localeCompare(b.name));
    };

    const reverseItems = () => {
      items.value.reverse();
    };

    const filterItems = () => {
      items.value = items.value.filter(item => item.name.length > 4);
    };

    const clearItems = () => {
      items.value = [];
    };

    return {
      items,
      newItem,
      addItem,
      removeItem,
      sortItems,
      reverseItems,
      filterItems,
      clearItems
    };
  }
}).mount('#app');
</script>

实际应用场景

列表渲染在实际开发中有很多应用场景：

常见应用场景

  • 待办事项列表：显示和管理任务项
  • 产品列表：电商网站的商品展示
  • 用户列表：管理系统中的用户展示
  • 评论列表：社交媒体中的评论展示
  • 导航菜单：动态生成导航项
  • 数据表格：展示结构化数据
实际应用示例 - 待办事项列表
<div id="app">
  <div class="todo-app">
    <h2>待办事项</h2>

    <div class="todo-form">
      <input v-model="newTodo" @keyup.enter="addTodo" placeholder="添加新任务">
      <button @click="addTodo">添加</button>
    </div>

    <div class="todo-stats">
      总数: {{ todos.length }} |
      已完成: {{ completedCount }} |
      未完成: {{ remainingCount }}
    </div>

    <ul class="todo-list">
      <li
        v-for="todo in todos"
        :key="todo.id"
        class="list-item"
        :class="{ completed: todo.completed }">
        <input
          type="checkbox"
          v-model="todo.completed">
        <span>{{ todo.text }}</span>
        <button @click="removeTodo(todo.id)">删除</button>
      </li>
    </ul>

    <div class="todo-actions">
      <button @click="removeCompleted">删除已完成</button>
      <button @click="toggleAll">{{ allDone ? '取消全选' : '全选' }}</button>
    </div>
  </div>
</div>

<script>
const { createApp, ref, computed } = Vue;

createApp({
  setup() {
    const newTodo = ref('');
    const todos = ref([
      { id: 1, text: '学习Vue.js', completed: false },
      { id: 2, text: '创建项目', completed: true },
      { id: 3, text: '部署应用', completed: false }
    ]);
    let nextId = 4;

    const addTodo = () => {
      if (newTodo.value.trim() !== '') {
        todos.value.push({
          id: nextId++,
          text: newTodo.value,
          completed: false
        });
        newTodo.value = '';
      }
    };

    const removeTodo = (id) => {
      todos.value = todos.value.filter(todo => todo.id !== id);
    };

    const removeCompleted = () => {
      todos.value = todos.value.filter(todo => !todo.completed);
    };

    const toggleAll = () => {
      const allDone = completedCount.value === todos.value.length;
      todos.value.forEach(todo => {
        todo.completed = !allDone;
      });
    };

    const completedCount = computed(() => {
      return todos.value.filter(todo => todo.completed).length;
    });

    const remainingCount = computed(() => {
      return todos.value.length - completedCount.value;
    });

    const allDone = computed(() => {
      return completedCount.value === todos.value.length && todos.value.length > 0;
    });

    return {
      newTodo,
      todos,
      addTodo,
      removeTodo,
      removeCompleted,
      toggleAll,
      completedCount,
      remainingCount,
      allDone
    };
  }
}).mount('#app');
</script>

性能优化建议

在使用列表渲染时，有一些性能优化的建议：

  • 始终为列表项提供key属性，使用唯一标识而不是index
  • 对于大型列表，考虑使用虚拟滚动技术
  • 避免在v-for中使用复杂的计算表达式，应使用计算属性代替
  • 使用v-show而不是v-if来切换列表项的可见性（如果切换频繁）
  • 限制列表渲染的数量，使用分页或无限滚动
