Vue.js简介与安装

Vue.js（/vjuː/，或简称为Vue）是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。

Vue.js核心特性

  • 渐进式框架：可以根据项目需要逐步采用Vue的功能
  • 双向数据绑定：通过v-model指令实现数据的双向绑定
  • 组件系统：将UI拆分为可复用的组件
  • 虚拟DOM：提高渲染性能
  • 轻量级：体积小，性能优秀
  • 易于学习：API设计简洁，学习曲线平缓

Vue.js的发展历程

Vue.js最初由尤雨溪开发，于2014年2月首次发布。经过几年的发展，Vue已经成为前端开发领域最受欢迎的框架之一。

版本 发布时间 主要特性
Vue 1.0 2015年10月 初始稳定版本，引入了响应式系统
Vue 2.0 2016年10月 引入虚拟DOM，组件系统得到完善
Vue 3.0 2020年9月 性能提升，引入组合式API
Vue 3.2 2022年6月 引入单文件组件语法糖

Vue.js与其他框架对比

Vue vs React
  • 学习曲线：Vue更易上手，React学习成本较高
  • 模板语法：Vue使用模板语法，React使用JSX
  • 生态系统：React社区更大，Vue更专注
  • 性能：两者性能接近，Vue 3性能更优
Vue vs Angular
  • 规模：Angular是全功能框架，Vue是渐进式框架
  • 学习难度：Vue更容易上手，Angular概念更复杂
  • 灵活性：Vue更灵活，Angular更规范
  • 依赖注入：Angular内置，Vue需要第三方库

安装Vue.js的方式

Vue.js提供了多种安装方式，你可以根据项目需求选择最适合的方式。

1. CDN引入（适合学习和小型项目）

最简单的方式是通过CDN引入Vue.js，适合初学者学习和小型项目。

<!-- 开发版本，包含完整的警告和调试功能 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- 生产版本，压缩且移除了警告 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>

2. NPM安装（推荐用于中大型项目）

在项目中使用NPM安装Vue.js，便于版本管理和依赖管理。

# 初始化项目
npm init -y

# 安装Vue
npm install vue

# 安装Vue CLI（可选，用于快速创建项目）
npm install -g @vue/cli

3. Vue CLI创建项目

使用Vue CLI可以快速搭建项目结构，集成开发工具。

# 创建新项目
vue create my-vue-project

# 进入项目目录
cd my-vue-project

# 启动开发服务器
npm run serve

4. Vite创建项目（推荐现代项目）

Vite是新一代前端构建工具，具有更快的冷启动和热更新速度。

# 使用Vite创建Vue项目
npm create vue@latest

# 或者使用yarn
yarn create vue

# 安装依赖
cd my-vue-project
npm install

# 启动开发服务器
npm run dev

第一个Vue应用

让我们创建一个简单的Vue应用来体验Vue的强大功能。

Hello Vue 示例
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Hello Vue</title>
</head>
<body>
  <div id="app">
    <h1>{{ title }}</h1>
    <p>欢迎来到{{ framework }}的世界！</p>
    <button @click="changeTitle">点击改变标题</button>
  </div>

  <!-- 开发版本 -->
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

  <script>
    const { createApp, ref } = Vue;

    createApp({
      setup() {
        const title = ref('Hello Vue!');
        const framework = ref('Vue.js');

        const changeTitle = () => {
          title.value = title.value === 'Hello Vue!' ? 'Hello World!' : 'Hello Vue!';
        };

        return {
          title,
          framework,
          changeTitle
        };
      }
    }).mount('#app');
  </script>
</body>
</html>

Vue.js生态系统

Vue.js拥有丰富的生态系统，包括官方和社区维护的各种工具和库：

  • Vue Router：官方路由管理器
  • Pinia/Vuex：官方状态管理库
  • Vue CLI：官方脚手架工具
  • Vite：现代化构建工具
  • Nuxt.js：服务端渲染框架
  • Vue DevTools：浏览器开发者工具
  • Vue Test Utils：官方测试工具
