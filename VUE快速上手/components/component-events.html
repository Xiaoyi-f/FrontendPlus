组件事件

组件事件是Vue.js中子组件向父组件传递数据和通信的重要方式。通过自定义事件，子组件可以向父组件发送消息，通知父组件发生了某些变化或需要执行某些操作。

组件事件核心概念

  • $emit方法：子组件触发自定义事件
  • v-on监听：父组件监听子组件事件
  • 事件参数：传递数据给父组件
  • 事件修饰符：处理特定事件行为
  • 事件验证：声明组件可以触发的事件
  • 事件总线：非父子组件间通信

基本组件事件

子组件通过$emit方法触发自定义事件，父组件通过v-on监听这些事件。

基本组件事件示例
<div id="app">
  <p>子组件计数: {{ childCount }}</p>
  <button-counter @increment="handleIncrement"></button-counter>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
    setup() {
      const childCount = ref(0);

      const handleIncrement = () => {
        childCount.value++;
      };

      return {
        childCount,
        handleIncrement
      };
    }
    components: {
      ButtonCounter: {
        data() {
          return {
            count: 0
          }
        },
        methods: {
          increment() {
            this.count++;
            this.$emit('increment');
          }
        },
        template: `
          <button @click="increment">点击: {{ count }}</button>
        `
      }
    }
}).mount('#app');
</script>

事件参数

子组件可以通过$emit方法传递参数给父组件，这些参数将作为事件处理函数的参数。

参数类型 说明 示例
基本类型 传递字符串、数字、布尔值等 $emit('event', 'value', 123, true)
对象 传递复杂数据结构 $emit('event', { id: 1, name: 'John' })
数组 传递多个值 $emit('event', [1, 2, 3, 4])
函数 传递回调函数 $emit('event', callbackFunction)
事件参数示例
<div id="app">
  <user-card
    v-for="user in users"
    :key="user.id"
    :user="user"
    @user-updated="updateUser"
    @user-deleted="deleteUser">
  </user-card>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
    setup() {
      const users = ref([
        { id: 1, name: '张三', email: 'zhangsan@example.com' },
        { id: 2, name: '李四', email: 'lisi@example.com' },
        { id: 3, name: '王五', email: 'wangwu@example.com' }
      ]);

      const updateUser = (userId, updatedData) => {
        console.log('更新用户', userId, updatedData);
        const userIndex = users.value.findIndex(u => u.id === userId);
        if (userIndex !== -1) {
          users.value[userIndex] = { ...users.value[userIndex], ...updatedData };
        }
      };

      const deleteUser = (userId) => {
        console.log('删除用户', userId);
        users.value = users.value.filter(u => u.id !== userId);
      };

      return {
        users,
        updateUser,
        deleteUser
      };
    }
    components: {
      UserCard: {
        props: ['user'],
        methods: {
          editUser() {
            this.$emit('user-updated', this.user.id, { name: '编辑后的用户' });
          },
          removeUser() {
            this.$emit('user-deleted', this.user.id);
          }
        },
        template: `
          <div class="user-card">
            <h3>{{ user.name }}</h3>
            <p>{{ user.email }}</p>
            <button @click="editUser">编辑</button>
            <button @click="removeUser">删除</button>
          </div>
        `
      }
    }
}).mount('#app');
</script>

事件修饰符

Vue提供了事件修饰符来处理常见的事件行为，这些修饰符同样适用于组件事件。

事件修饰符示例
<div id="app">
  <form-component
    @submit.prevent="handleSubmit"
    @submit.once="handleOnceSubmit">
  </form-component>
</div>

<script>
const { createApp } = Vue;

createApp({
    setup() {
      const handleSubmit = () => {
        console.log('表单提交被阻止默认行为');
      };

      const handleOnceSubmit = () => {
        console.log('仅触发一次的提交事件');
      };

      return {
        handleSubmit,
        handleOnceSubmit
      };
    }
    components: {
      FormComponent: {
        template: `
          <form>
            <input type="text" placeholder="输入数据">
            <button type="submit">提交</button>
          </form>
        `,
        mounted() {
          this.$emit('submit');
        }
      }
    }
}).mount('#app');
</script>

使用emits选项

Vue 3中可以使用emits选项声明组件会触发的事件，这有助于提高代码的可读性和维护性。

emits选项示例
<div id="app">
  <counter-component
    :initial-value="10"
    @count-changed="handleCountChange"
    @reset="handleReset">
  </counter-component>
</div>

<script>
const { createApp, ref } = Vue;

createApp({
    setup() {
      const message = ref('');

      const handleCountChange = (newCount) => {
        message.value = `计数改变为: ${newCount}`;
      };

      const handleReset = () => {
        message.value = '计数器已重置';
      };

      return {
        message,
        handleCountChange,
        handleReset
      };
    }
    components: {
      CounterComponent: {
        props: ['initialValue'],
        emits: ['count-changed', 'reset'],
        data() {
          return {
            count: this.initialValue
          }
        },
        methods: {
          increment() {
            this.count++;
            this.$emit('count-changed', this.count);
          },
          decrement() {
            this.count--;
            this.$emit('count-changed', this.count);
          },
          reset() {
            this.count = this.initialValue;
            this.$emit('reset');
          }
        },
        template: `
          <div>
            <p>计数: {{ count }}</p>
            <button @click="increment">+</button>
            <button @click="decrement">-</button>
            <button @click="reset">重置</button>
          </div>
        `
      }
    }
}).mount('#app');
</script>

事件与组合式API

在组合式API中，事件的处理方式略有不同，但核心概念保持一致。

选项式API
const ChildComponent = {
  emits: ['update-value', 'click'],
  methods: {
    updateValue(newValue) {
      this.$emit('update-value', newValue);
    }
  }
};
组合式API
const ChildComponent = {
  emits: ['update-value', 'click'],
  setup(props, { emit }) {
    const updateValue = (newValue) => {
      emit('update-value', newValue);
    };

    return {
      updateValue
    };
  }
};

实际应用场景

组件事件在实际开发中有很多应用场景：

常见应用场景

  • 表单组件：提交表单、验证错误、值变更
  • 按钮组件：点击、双击、长按等事件
  • 列表组件：选择、删除、编辑等操作
  • 模态框组件：打开、关闭、确认等事件
  • 输入组件：值改变、失焦、输入等事件
  • 卡片组件：展开、收起、操作等事件
实际应用示例 - 购物车组件
<div id="app">
  <div class="cart-summary">
    <h2>购物车 ({{ cartItems.length }} 件商品)</h2>
    <p>总计: ¥{{ cartTotal.toFixed(2) }}</p>
  </div>
  <cart-item
    v-for="item in cartItems"
    :key="item.id"
    :item="item"
    @quantity-changed="updateQuantity"
    @item-removed="removeItem">
  </cart-item>
</div>

<script>
const { createApp, ref, computed } = Vue;

createApp({
    setup() {
      const cartItems = ref([
        { id: 1, name: 'iPhone 13', price: 6999, quantity: 1 },
        { id: 2, name: 'MacBook Pro', price: 12999, quantity: 1 },
        { id: 3, name: 'iPad Air', price: 4399, quantity: 2 }
      ]);

      const cartTotal = computed(() => {
        return cartItems.value.reduce((total, item) => {
          return total + (item.price * item.quantity);
        }, 0);
      });

      const updateQuantity = (itemId, newQuantity) => {
        const item = cartItems.value.find(i => i.id === itemId);
        if (item) {
          item.quantity = newQuantity;
        }
      };

      const removeItem = (itemId) => {
        cartItems.value = cartItems.value.filter(item => item.id !== itemId);
      };

      return {
        cartItems,
        cartTotal,
        updateQuantity,
        removeItem
      };
    }
    components: {
      CartItem: {
        props: ['item'],
        emits: ['quantity-changed', 'item-removed'],
        methods: {
          updateQuantity(newQuantity) {
            this.$emit('quantity-changed', this.item.id, newQuantity);
          },
          removeItem() {
            this.$emit('item-removed', this.item.id);
          }
        },
        template: `
          <div class="cart-item">
            <h4>{{ item.name }}</h4>
            <p>单价: ¥{{ item.price }}</p>
            <div class="quantity-control">
              <label>数量:</label>
              <input
                type="number"
                :value="item.quantity"
                min="1"
                @input="updateQuantity($event.target.value)">
            </div>
            <p>小计: ¥{{ (item.price * item.quantity).toFixed(2) }}</p>
            <button @click="removeItem">删除</button>
          </div>
        `
      }
    }
}).mount('#app');
</script>

最佳实践建议

在使用组件事件时，有一些最佳实践建议：

  • 使用emits选项声明组件会触发的事件，提高代码可读性
  • 事件名称使用kebab-case格式（短横线分隔）
  • 为事件传递有意义的参数
  • 避免在事件处理中进行复杂计算，尽量只做状态更新
  • 合理使用事件修饰符
  • 对于复杂的组件通信，考虑使用状态管理工具

实践练习

  • 创建一个评论组件，允许用户提交评论并通过事件通知父组件
  • 创建一个评分组件，用户点击星星评分并通过事件传递评分值
  • 创建一个选项卡组件，切换标签页时触发事件
  • 创建一个文件上传组件，上传时触发进度和完成事件
  • 创建一个搜索组件，输入时触发搜索事件
  • 创建一个分页组件，页码改变时触发事件
