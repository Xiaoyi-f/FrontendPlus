什么是组件？

组件是Vue.js最强大的功能之一。组件可以扩展HTML元素，封装可重用的代码。在较高层面上，组件是自定义元素，Vue.js的编译器为它添加特殊功能。在有些情况下，组件也可以表现为用[is](file:///e:\FrontendTeach\VUE快速上手\basics\template-syntax.html#L37-L37)特性控制的原生HTML元素。

组件注册

组件在使用前需要先注册。注册有两种方式：全局注册和局部注册。

全局注册

全局注册的组件可以在任何Vue实例中使用：

// 全局注册组件 Vue.component('my-component', { template: '<div>这是一个全局组件</div>' }) // 现在可以在任何Vue实例的模板中使用 const app = new Vue({ el: '#app' })

局部注册

局部注册的组件只能在当前实例或组件中使用：

const ComponentA = { template: '<div>这是局部组件A</div>' } const ComponentB = { template: '<div>这是局部组件B</div>' } const app = new Vue({ el: '#app', components: { 'component-a': ComponentA, 'component-b': ComponentB } })

使用组件

组件可以通过自定义标签在模板中使用：

<!-- HTML --> <div id="app"> <my-component></my-component> <component-a></component-a> <component-b></component-b> </div>

组件命名规范

Vue组件的命名遵循以下规范：

  • 使用PascalCase或kebab-case：推荐使用PascalCase（驼峰命名法）
  • 语义化命名：组件名称应具有明确的语义
  • 避免HTML标签名：不要使用已存在的HTML元素名称
// 推荐的命名方式 Vue.component('MyButton', { /* ... */ }) Vue.component('UserCard', { /* ... */ }) // 或者使用kebab-case Vue.component('my-button', { /* ... */ }) Vue.component('user-card', { /* ... */ })

组件嵌套

组件可以相互嵌套，形成组件树结构：

// 定义一个按钮组件 const Button = { template: `<button class="my-button">{{ label }}</button>`, props: ['label'] } // 定义一个卡片组件，包含按钮 const Card = { template: ` <div class="card"> <h3>{{ title }}</h3> <p>{{ content }}</p> <button-component :label="buttonLabel"></button-component> </div> `, props: ['title', 'content', 'buttonLabel'], components: { 'button-component': Button } } // 定义根组件，包含卡片 const App = { template: ` <div class="app"> <card-component title="Vue组件示例" content="这是一个组件嵌套的示例" button-label="点击我" ></card-component> </div> `, components: { 'card-component': Card } } const app = new Vue({ el: '#app', components: { 'app-component': App } })

组件的复用

组件最重要的特性之一是复用。通过props可以向组件传递不同的数据：

// 定义一个用户卡片组件 const UserCard = { template: ` <div class="user-card"> <h3>{{ user.name }}</h3> <p>{{ user.email }}</p> <p>{{ user.role }}</p> </div> `, props: ['user'] } // 在不同地方使用同一组件，传入不同的数据 const app = new Vue({ el: '#app', data: { users: [ { id: 1, name: '张三', email: 'zhangsan@example.com', role: '管理员' }, { id: 2, name: '李四', email: 'lisi@example.com', role: '用户' }, { id: 3, name: '王五', email: 'wangwu@example.com', role: '编辑' } ] }, components: { 'user-card': UserCard } })

完整示例

下面是一个完整的组件基础示例：

<!DOCTYPE html> <html> <head> <title>Vue组件示例</title> <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script> </head> <body> <div id="app"> <product-card v-for="product in products" :key="product.id" :product="product" ></product-card> </div> <script> // 定义产品卡片组件 Vue.component('product-card', { template: ` <div class="product-card"> <h3>{{ product.name }}</h3> <p class="price">¥{{ product.price }}</p> <p class="description">{{ product.description }}</p> <button @click="buyProduct">购买</button> </div> `, props: ['product'], methods: { buyProduct() { alert(`您选择了购买: ${this.product.name}`) } } }) new Vue({ el: '#app', data: { products: [ { id: 1, name: 'iPhone 13', price: 6999, description: '最新款iPhone' }, { id: 2, name: 'MacBook Pro', price: 12999, description: '专业笔记本' }, { id: 3, name: 'iPad Air', price: 4399, description: '轻薄平板' } ] } }) </script> <style> .product-card { border: 1px solid #ddd; border-radius: 8px; padding: 15px; margin: 10px; width: 250px; display: inline-block; vertical-align: top; } .price { color: #e74c3c; font-weight: bold; font-size: 1.2em; } .description { color: #7f8c8d; } </style> </body> </html>

组件的data选项

组件的[data](file:///e:\FrontendTeach\VUE快速上手\basics\instance.html#L20-L20)选项必须是一个函数，这样每个组件实例才能维护一份被返回对象的独立的拷贝：

// 正确的写法 const Component = { data: function() { return { count: 0 } } } // 错误的写法 const Component = { data: { count: 0 } }

组件的组织结构

在大型应用中，通常使用构建工具和单文件组件（.vue文件）来组织组件：

// ProductCard.vue <template> <div class="product-card"> <h3>{{ product.name }}</h3> <p class="price">¥{{ product.price }}</p> <p class="description">{{ product.description }}</p> <button @click="buyProduct">购买</button> </div> </template> <script> export default { name: 'ProductCard', props: { product: { type: Object, required: true } }, methods: { buyProduct() { alert(`您选择了购买: ${this.product.name}`) } } } </script> <style scoped> .product-card { border: 1px solid #ddd; border-radius: 8px; padding: 15px; margin: 10px; width: 250px; display: inline-block; vertical-align: top; } .price { color: #e74c3c; font-weight: bold; font-size: 1.2em; } .description { color: #7f8c8d; } </style>
