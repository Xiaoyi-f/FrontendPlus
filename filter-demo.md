filter-demo.html中使用的滤镜讲解
1. 模糊滤镜 blur —— 给元素 “磨皮”
语法：filter: blur(数值px);
参数：数值越大，模糊越明显（不能为负数）；
代码示例：filter: blur(2px); → 图片轻微模糊，像加了一层雾；
常用场景：毛玻璃效果、背景模糊、弱化次要元素。
2. 亮度滤镜 brightness —— 调亮 / 调暗元素
语法：filter: brightness(数值);
参数规则：
1（或 100%）= 原图亮度；
大于 1（如 1.3/130%）= 变亮；
小于 1（如 0.8/80%）= 变暗；
代码示例：filter: brightness(1.3); → 图片亮度提升 30%；
常用场景：hover 时提亮图片、修复偏暗的图片。
3. 对比度滤镜 contrast —— 强化明暗差异
语法：filter: contrast(数值/百分比);
参数规则：
1（100%）= 原图对比度；
大于 1（如 1.5/150%）= 对比更强（亮的更亮、暗的更暗）；
小于 1 = 对比减弱（画面变灰）；
代码示例：filter: contrast(150%); → 对比度提升 50%，画面更通透；
常用场景：突出图片细节、黑白图片强化对比。
4. 灰度滤镜 grayscale —— 把彩色变黑白
语法：filter: grayscale(数值/百分比);
参数规则：
0（0%）= 原图（彩色）；
1（100%）= 完全灰度（黑白）；
0~1 之间 = 半灰度（比如 0.5=50% 灰度）；
代码示例：filter: grayscale(100%); → 图片变成纯黑白；
常用场景：黑白老照片效果、hover 时恢复彩色、哀悼页全站置灰。
5. 色相旋转 hue-rotate —— 给图片 “换色”
语法：filter: hue-rotate(角度deg);
参数：角度范围 0~360deg，不同角度对应不同色相（比如 90deg 偏绿、180deg 偏蓝）；
代码示例：filter: hue-rotate(90deg); → 图片整体色相旋转 90 度，彩色图片变 “变色版”；
常用场景：快速换主题色、创意色彩效果、批量改图片色调。
6. 反色滤镜 invert —— 像 “底片” 一样反转颜色
语法：filter: invert(数值/百分比);
参数规则：
0（0%）= 原图；
1（100%）= 完全反色（黑变白、红变绿、蓝变黄）；
代码示例：filter: invert(100%); → 图片变成底片效果；
常用场景：暗黑模式快速适配、创意视觉特效。
7. 透明度滤镜 opacity —— 让元素 “变透明”
语法：filter: opacity(数值);
参数规则：
1（100%）= 完全不透明；
0（0%）= 完全透明；
0~1 之间 = 半透明（比如 0.7=70% 不透明）；
代码示例：filter: opacity(0.7); → 图片 70% 不透明，30% 透明；
❗ 注意：和 CSS 的 opacity 属性效果类似，但 filter: opacity 是滤镜，性能略差，新手优先用 opacity 属性（opacity: 0.7;）。
8. 饱和度滤镜 saturate —— 调色彩鲜艳度
语法：filter: saturate(数值/百分比);
参数规则：
1（100%）= 原图饱和度；
大于 1（如 2/200%）= 色彩更鲜艳；
小于 1 = 色彩更淡（接近灰度）；
代码示例：filter: saturate(200%); → 饱和度翻倍，图片色彩更浓；
常用场景：风景图提亮色彩、hover 时增强饱和度。
9. 褐色滤镜 sepia —— 给图片加 “复古泛黄” 效果
语法：filter: sepia(数值/百分比);
参数规则：
0（0%）= 原图；
1（100%）= 完全褐色（老照片质感）；
代码示例：filter: sepia(80%); → 80% 褐色效果，图片偏黄，有复古感；
常用场景：复古风照片、怀旧主题设计。
10. 投影滤镜 drop-shadow —— 给元素加 “彩色阴影”
语法：filter: drop-shadow(水平偏移px 垂直偏移px 模糊px 颜色);
参数拆解：
水平偏移：正数右移，负数左移；
垂直偏移：正数下移，负数上移；
模糊：数值越大，阴影越虚（0 = 无模糊）；
颜色：可以用 rgba（带透明度）、十六进制等；
代码示例：drop-shadow(0 4px 6px rgba(255, 100, 100, 0.5)) → 红色半透明阴影，下移 4px、模糊 6px；
❗ 对比 box-shadow：drop-shadow 会跟随元素的 “形状”（比如圆形图片），而 box-shadow 是矩形阴影，更适合不规则元素。
11. 组合滤镜 —— 实战最常用！
核心逻辑：多个滤镜用空格分隔，按顺序生效；
代码示例：filter: blur(0.5px) brightness(1.1) contrast(110%) sepia(20%) saturate(130%);
效果：轻微模糊 + 提亮 + 增强对比 + 轻微复古 + 增强饱和度 → 调出 “复古清新” 的质感；
技巧：新手可以先调 brightness/contrast/saturate 基础款，再叠加 blur/sepia 做细节
