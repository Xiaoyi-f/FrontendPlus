ES6+新特性

ES6（ECMAScript 2015）是JavaScript语言的重大更新，引入了许多现代编程特性。ES6+则包括后续版本（ES2016、ES2017、ES2018等）引入的新功能。掌握这些特性对于现代JavaScript开发至关重要。

ES6+新特性概览

  • let/const：块级作用域变量声明
  • 解构赋值：从对象或数组中提取值
  • 模板字符串：更简洁的字符串拼接
  • 箭头函数：更简洁的函数语法
  • 类：基于原型的语法糖
  • 模块：原生模块系统
  • Promise：异步编程解决方案
  • async/await：更优雅的异步处理
  • 扩展运算符：展开对象或数组
  • 默认参数：函数参数的默认值
  • 解构赋值：更简洁的数据提取方式
  • 生成器：函数的特殊执行方式

变量声明：let 和 const

ES6引入了let和const，提供了块级作用域，解决了var声明的变量提升和作用域问题。

// var声明的问题
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(`var: ${i}`), 100);
}
// 输出: 3, 3, 3

// 使用let修复
for (let j = 0; j < 3; j++) {
  setTimeout(() => console.log(`let: ${j}`), 200);
}
// 输出: 0, 1, 2

// const声明常量
const PI = 3.14159;
// PI = 3; // TypeError: Assignment to constant variable.

// const声明对象
const person = { name: "Alice", age: 25 };
person.name = "Bob"; // 可以修改属性
person.newProp = "new"; // 可以添加属性
// person = {}; // TypeError: Assignment to constant variable.

解构赋值

解构赋值允许从数组或对象中提取值，创建变量，使代码更简洁易读。

// 数组解构
const colors = ['red', 'green', 'blue'];
const [firstColor, secondColor, thirdColor] = colors;
console.log(firstColor, secondColor, thirdColor); // 'red' 'green' 'blue'

// 跳过元素
const [a, , c] = [1, 2, 3];
console.log(a, c); // 1 3

// 对象解构
const user = { name: 'John', age: 30, city: 'New York' };
const { name, age } = user;
console.log(name, age); // 'John' 30

// 重命名
const { name: userName, age: userAge } = user;
console.log(userName, userAge); // 'John' 30

// 默认值
const { name = 'Anonymous', country = 'Unknown' } = { name: 'Alice' };
console.log(name, country); // 'Alice' 'Unknown'

// 函数参数解构
function displayUser({ name = 'Anonymous', age = 0 }) {
  console.log(`Name: ${name}, Age: ${age}`);
}
displayUser({ name: 'Bob' }); // 'Name: Bob, Age: 0'

模板字符串

模板字符串使用反引号定义，支持多行文本和表达式插值。

// 基本模板字符串
const name = "Alice";
const greeting = `Hello, ${name}!`;
console.log(greeting); // 'Hello, Alice!'

// 多行字符串
const multiline = `This is line 1 This is line 2 This is line 3`;
console.log(multiline);

// 表达式插值
const a = 5;
const b = 10;
const result = `The sum of ${a} and ${b} is ${a + b}`;
console.log(result); // 'The sum of 5 and 10 is 15'

// 嵌套表达式
const user = { name: 'John', age: 30 };
const userInfo = `   User Information:   Name: ${user.name}   Age: ${user.age}   Adult: ${user.age >= 18 ? 'Yes' : 'No'} `;
console.log(userInfo);

箭头函数

箭头函数提供了更简洁的函数语法，并且this的绑定方式与传统函数不同。

// 传统函数 vs 箭头函数
const add = function(a, b) {
  return a + b;
};

const addArrow = (a, b) => a + b; // 简化写法
const square = x => x * x; // 单参数可省略括号
const greet = () => 'Hello!'; // 无参数需要空括号

// 返回对象需要括号
const createUser = (name, age) => ({ name, age });

// 箭头函数中的this
function Timer() {
  this.seconds = 0;

  setInterval(() => {
    // 箭头函数保持this指向Timer实例
    this.seconds++;
    console.log(this.seconds);
  }, 1000);
}

// 传统函数vs箭头函数this的差异
const obj = {
  name: 'Object',
  traditionalFunc: function() {
    console.log('Traditional: ' + this.name); // 'Object'
  },
  arrowFunc: () => {
    console.log('Arrow: ' + this.name); // undefined (指向全局作用域)
  }
};
箭头函数特性
  • 简洁语法：更少的代码量
  • 隐式返回：单表达式自动返回
  • this绑定：继承外层作用域的this
  • 无arguments对象：需使用剩余参数
  • 不能用作构造函数：无new.target
何时使用箭头函数
  • 数组方法：map, filter, reduce等
  • 回调函数：保持this上下文
  • 简短函数：单行逻辑
  • 避免使用：对象方法、构造函数、需要动态this的场景

类（Classes）

ES6引入了类语法，提供更清晰的面向对象编程方式，但底层仍是基于原型的。

class Animal {
  constructor(name, species) {
    this.name = name;
    this.species = species;
  }

  makeSound() {
    console.log(`${this.name} makes a sound.`);
  }

  getInfo() {
    return `${this.name} is a ${this.species}`;
  }

  static getKingdom() {
    return "Animalia";
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name, "Dog"); // 调用父类构造函数
    this.breed = breed;
  }

  makeSound() {
    console.log(`${this.name} barks!`);
  }

  fetch() {
    console.log(`${this.name} fetches the ball!`);
  }
}

const myDog = new Dog("Buddy", "Golden Retriever");
myDog.makeSound(); // "Buddy barks!"
console.log(myDog.getInfo()); // "Buddy is a Dog"
myDog.fetch(); // "Buddy fetches the ball!"
console.log(Dog.getKingdom()); // "Animalia"

模块化

ES6模块系统允许我们组织代码到不同的文件中，并导出/导入功能。

// math.js - 导出模块
// 默认导出
export default function add(a, b) {
  return a + b;
}

// 命名导出
export const PI = 3.14159;

export function multiply(a, b) {
  return a * b;
}

export class Calculator {
  constructor() {
    this.history = [];
  }

  add(a, b) {
    const result = a + b;
    this.history.push({ operation: 'add', a, b, result });
    return result;
  }
}

// main.js - 导入模块
// 导入默认导出
import add from './math.js';

// 导入命名导出
import { PI, multiply, Calculator } from './math.js';

// 导入所有命名导出
import * as mathUtils from './math.js';

// 混合导入
import add, { PI, multiply } from './math.js';

// 重命名导入
import { multiply as mult } from './math.js';
ES2016+新特性
// Array.prototype.includes() - ES2016
const arr = [1, 2, 3];
console.log(arr.includes(2)); // true
console.log(arr.includes(4)); // false

// 指数运算符 - ES2016
console.log(2 ** 3); // 8
console.log(Math.pow(2, 3)); // 8 (等价于 2 ** 3)

// Object.values() / Object.entries() - ES2017
const obj = { a: 1, b: 2, c: 3 };
console.log(Object.values(obj)); // [1, 2, 3]
console.log(Object.entries(obj)); // [['a', 1], ['b', 2], ['c', 3]]

// padStart() / padEnd() - ES2017
console.log('5'.padStart(3, '0')); // '005'
console.log('5'.padEnd(3, '0')); // '500'

// Object.getOwnPropertyDescriptors() - ES2017
const obj2 = {
  get name() { return 'John'; }
};
console.log(Object.getOwnPropertyDescriptors(obj2));

// async/await - ES2017
async function fetchData() {
  try {
    const response = await fetch('/api/data');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error:', error);
  }
}

// ES2022 - at() 方法
const arr2 = [1, 2, 3, 4, 5];
console.log(arr2.at(0)); // 1
console.log(arr2.at(-1)); // 5 (最后一个元素)

// ES2023 - findLast() 和 findLastIndex()
const numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];
console.log(numbers.findLast(n => n % 2 === 0)); // 2 (从末尾开始找到的第一个偶数)
console.log(numbers.findLastIndex(n => n % 2 === 0)); // 7 (从末尾开始找到的第一个偶数的索引)

剩余参数和扩展运算符

剩余参数允许我们将不定数量的参数收集到数组中，扩展运算符则将数组展开。

// 剩余参数
function sum(...numbers) {
  return numbers.reduce((total, num) => total + num, 0);
}
console.log(sum(1, 2, 3, 4, 5)); // 15

// 扩展运算符
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const combined = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6]

// 复制数组
const originalArray = [1, 2, 3];
const copiedArray = [...originalArray];

// 对象扩展
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };
const mergedObj = { ...obj1, ...obj2 }; // {a: 1, b: 2, c: 3, d: 4}

// 函数调用时使用扩展运算符
const numbers = [1, 5, 3, 2, 4];
console.log(Math.max(...numbers)); // 5

生成器（Generators）

生成器函数可以暂停执行并在稍后恢复，提供了一种处理异步操作和创建迭代器的方式。

// 基本生成器函数
function* simpleGenerator() {
  yield 1;
  yield 2;
  yield 3;
}

const gen = simpleGenerator();
console.log(gen.next()); // {value: 1, done: false}
console.log(gen.next()); // {value: 2, done: false}
console.log(gen.next()); // {value: 3, done: false}
console.log(gen.next()); // {value: undefined, done: true}

// 使用生成器创建斐波那契数列
function* fibonacci() {
  let a = 0, b = 1;
  while (true) {
    yield a;
    [a, b] = [b, a + b]; // 解构赋值
  }
}

const fib = fibonacci();
console.log(fib.next().value); // 0
console.log(fib.next().value); // 1
console.log(fib.next().value); // 1
console.log(fib.next().value); // 2
console.log(fib.next().value); // 3
console.log(fib.next().value); // 5
