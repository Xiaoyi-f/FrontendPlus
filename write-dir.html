1. 默认书写方向（左→右）
逻辑属性对应：左/下/左右/左边框
margin-inline-start: 10px → 左外边距
margin-block-end: 15px → 下外边距
padding-inline: 10px 20px → 左内边距10px，右内边距20px
border-inline-start: 2px solid red → 左边框（红）
2. 书写方向（右→左）
逻辑属性对应：右/下/右左/右边框（代码和上面完全一样！）
margin-inline-start: 10px → 右外边距
margin-block-end: 15px → 下外边距（不变）
padding-inline: 10px 20px → 右内边距10px，左内边距20px
border-inline-start: 2px solid red → 右边框（红）
3. 垂直书写方向（上→下）
逻辑属性对应：上/左/上下/上边框（代码仍不变！）
margin-inline-start: 10px → 上外边距
margin-block-end: 15px → 左外边距
padding-inline: 10px 20px → 上内边距10px，下内边距20px
border-inline-start: 2px solid red → 上边框（红）
4. 物理属性（固定左/下）
margin-left/ border-left 永远是左边，不随书写方向变
margin-left: 10px → 永远左外边距
margin-bottom: 15px → 永远下外边距
padding-left: 10px → 永远左内边距
border-left: 2px solid red → 永远左边框
