CSS Filter 滤镜 一站式综合学习（hover看原图）

blur(2px) → 模糊效果

参数：像素值（越大越模糊）
场景：毛玻璃、背景朦胧、氛围感

brightness(1.3) → 亮度提升

参数：1=原图，>1变亮/<1变暗
场景：高光、暗角、hover提亮

contrast(150%) → 对比度增强

参数：100%=原图，>100%更通透
场景：提升图片质感、黑白强化

grayscale(100%) → 纯黑白

参数：0%-100%，100%全灰
场景：黑白照片、网站置灰、hover变色

hue-rotate(90deg) → 色相旋转

参数：角度（0-360deg）
场景：调整主色调、多彩滤镜

invert(100%) → 完全反色

参数：0%-100%，100%负片效果
场景：负片、深色模式快速适配

opacity(0.7) → 70%不透明

区别于opacity属性：支持SVG+组合滤镜
场景：半透明、SVG透明处理

saturate(200%) → 饱和度翻倍

参数：0%无色彩，>100%更鲜艳
场景：提升色彩、低饱和高级感

sepia(80%) → 复古褐色

参数：0%-100%，100%纯褐色
场景：老照片、怀旧风格

drop-shadow(0 4px 6px 红)

区别于box-shadow：只给非透明区加阴影
场景：透明PNG图标、不规则元素投影

组合滤镜 → 复古清新风

blur+brightness+contrast+sepia+saturate
场景：定制图片滤镜、网红效果

hover交互 → 黑白变彩色

默认：grayscale+brightness
hover：grayscale(0)+提亮+饱和
