HTML表单类型大纲
## 一、核心`<input>`系列（极致细分，含所有type值及变体）
### （一）基础输入类（含小众变体）
1.  普通单行文本框（type="text"）
2.  带占位符文本框（type="text" + placeholder属性）
3.  只读单行文本框（type="text" + readonly属性）
4.  禁用单行文本框（type="text" + disabled属性）
5.  带长度限制文本框（type="text" + maxlength/minlength属性）
6.  必填单行文本框（type="text" + required属性）
7.  密码输入框（type="password"）
8.  显示明文密码框（type="password" + toggle切换（JS辅助））
9.  带密码强度提示框（type="password" + JS联动反馈）
10. 隐藏域（type="hidden"）
11. 搜索输入框（type="search"）
12. 带清除按钮搜索框（type="search"，部分浏览器原生支持）
13. 电话输入框（type="tel"）
14. 带手机号验证电话框（type="tel" + pattern属性）
15. 邮箱输入框（type="email"）
16. 多邮箱输入框（type="email" + multiple属性，用逗号分隔多个邮箱）
17. 网址输入框（type="url"）
18. 带协议提示网址框（type="url" + placeholder="https://"）

### （二）数值/范围类（全变体覆盖）
19. 数字输入框（type="number"）
20. 带范围限制数字框（type="number" + min/max/step属性）
21. 整数数字框（type="number" + step="1"）
22. 小数数字框（type="number" + step="0.1/0.01"等）
23. 禁用增减按钮数字框（type="number" + CSS隐藏箭头）
24. 滑块控件（type="range"）
25. 带数值反馈滑块（type="range" + <output>联动）
26. 多区间滑块（type="range" + JS扩展双滑块/多滑块）
27. 带刻度滑块（type="range" + CSS自定义刻度）

### （三）日期/时间类（含所有原生type及兼容性变体）
28. 日期选择器（type="date"，年月日）
29. 本地日期时间选择器（type="datetime-local"，年月日+时分）
30. 月份选择器（type="month"，年+月）
31. 周选择器（type="week"，年+周数）
32. 时间选择器（type="time"，时分/时分秒）
33. 带秒数时间选择器（type="time" + step="1"，显示秒数）
34. 日期时间选择器（type="datetime"，含时区，兼容性差）
35. 自定义样式日期选择器（原生type="date" + CSS覆盖默认样式）
36. 兼容低版本浏览器日期框（文本框 + JS日历插件，如laydate/flatpickr）

### （四）选择类（全细分类型）
37. 单选按钮（type="radio"）
38. 禁用单选按钮（type="radio" + disabled属性）
39. 默认选中单选按钮（type="radio" + checked属性）
40. 分组单选按钮（同name属性的radio组）
41. 复选框（type="checkbox"）
42. 禁用复选框（type="checkbox" + disabled属性）
43. 默认选中复选框（type="checkbox" + checked属性）
44. 全选/反选复选框组（JS联动的checkbox组）
45. 不确定状态复选框（type="checkbox" + indeterminate属性，半选中状态）
46. 颜色选择器（type="color"）
47. 预设颜色选择器（type="color" + JS限制可选颜色）

### （五）功能按钮类（所有变体）
48. 普通按钮（type="button"）
49. 禁用普通按钮（type="button" + disabled属性）
50. 带图标普通按钮（type="button" + 图标标签/背景图）
51. 提交按钮（type="submit"）
52. 禁用提交按钮（type="submit" + disabled属性）
53. 自定义样式提交按钮（type="submit" + CSS美化）
54. 重置按钮（type="reset"）
55. 禁用重置按钮（type="reset" + disabled属性）
56. 图像提交按钮（type="image"，图片作为提交按钮）
57. 带点击坐标反馈图像按钮（type="image"，提交时传递x/y坐标参数）

### （六）文件上传类（极致细分）
58. 单文件上传框（type="file"）
59. 多文件上传框（type="file" + multiple属性）
60. 指定类型文件上传框（type="file" + accept属性，如image/*/.pdf/.docx）
61. 图片上传预览框（type="file" + accept="image/*" + JS预览图片）
62. 禁用文件上传框（type="file" + disabled属性）
63. 自定义样式文件上传框（隐藏原生控件 + 模拟按钮/预览区）
64. 大文件分片上传框（type="file" + JS分片处理）

## 二、独立表单元素（全类型+细分变体）
### （一）文本输入类
65. 普通多行文本域（`<textarea>`）
66. 带行数列数限制文本域（`<textarea>` + rows/cols属性）
67. 可调整大小文本域（`<textarea>`，默认resize: both）
68. 仅横向调整文本域（`<textarea>` + resize: horizontal）
69. 仅纵向调整文本域（`<textarea>` + resize: vertical）
70. 不可调整大小文本域（`<textarea>` + resize: none）
71. 只读文本域（`<textarea>` + readonly属性）
72. 禁用文本域（`<textarea>` + disabled属性）
73. 带长度限制文本域（`<textarea>` + maxlength/minlength属性）
74. 必填文本域（`<textarea>` + required属性）
75. 带滚动条文本域（`<textarea>` + overflow: auto/scroll）
76. 富文本编辑器（非原生，JS实现：CKEditor/TinyMCE/Quill，支持图文/格式化输入）

### （二）下拉选择类（全形态）
77. 普通单选下拉框（`<select>` + `<option>`）
78. 多选下拉框（`<select>` + `<option>` + multiple属性）
79. 分组下拉框（`<select>` + `<optgroup>` + `<option>`）
80. 可搜索下拉框（`<select>` + JS插件，如select2/element-ui Select）
81. 带禁用选项下拉框（`<option>` + disabled属性）
82. 带禁用分组下拉框（`<optgroup>` + disabled属性）
83. 默认选中下拉框（`<option>` + selected属性）
84. 带默认占位符下拉框（`<option value="">请选择</option>`）
85. 可编辑下拉框（支持输入+选择，JS扩展，如datalist增强/第三方插件）
86. 级联下拉框（联动下拉，如省-市-区，JS/ajax联动）
87. 禁用下拉框（`<select>` + disabled属性）

### （三）按钮类（增强版全变体）
88. 普通增强按钮（`<button type="button">`）
89. 提交增强按钮（`<button type="submit">`）
90. 重置增强按钮（`<button type="reset">`）
91. 禁用增强按钮（`<button>` + disabled属性）
92. 带图标增强按钮（`<button>` + 图标元素/字体图标）
93. 带加载状态增强按钮（`<button>` + JS切换加载动画）
94. 悬浮样式增强按钮（`<button>` + :hover样式）
95. 点击反馈增强按钮（`<button>` + :active样式/JS点击效果）

## 三、HTML5特殊表单控件（全功能覆盖）
96. 表单字段集（`<fieldset>`，分组表单元素）
97. 禁用字段集（`<fieldset>` + disabled属性，组内控件全部禁用）
98. 字段集标题（`<legend>`，配合`<fieldset>`标注分组）
99. 标签关联（`<label>`）
    - 隐式关联（`<label for="控件ID">` + 控件id）
    - 显式关联（`<label>`直接包裹表单控件）
100. 输出控件（`<output>`，显示计算结果/滑块反馈）
101. 数据列表（`<datalist>` + `<option>`，输入框下拉建议）
102. 进度条（`<progress>`）
    - 确定进度条（`<progress>` + value/max属性）
    - 不确定进度条（`<progress>`，无value属性，加载动画）
103. 度量条（`<meter>`，显示度量值）
    - 带阈值度量条（`<meter>` + min/max/low/high/optimum属性）
    - 不同状态度量条（根据值自动切换颜色，如低/中/高）

## 四、扩展表单类型（现代开发常用，原生+第三方）
104. 单选框组（美化版，隐藏原生radio + CSS模拟样式）
105. 复选框组（美化版，隐藏原生checkbox + CSS模拟样式）
106. 开关按钮（表单变体，type="checkbox" + CSS模拟开关状态）
107. 星级评分控件（表单变体，radio/checkbox + CSS/JS实现星级选择）
108. 日期范围选择器（JS扩展，如daterangepicker，选择开始/结束日期）
109. 时间范围选择器（JS扩展，选择开始/结束时间）
110. 拖拽上传表单（文件上传变体，支持拖拽文件到指定区域上传）
111. 批量上传表单（多文件上传 + 队列管理/断点续传）
112. 验证码输入框（文本框 + 图形验证码/短信验证码/滑块验证码）
113. 身份证输入框（type="text" + pattern/JS验证身份证格式）
114. 银行卡输入框（type="text" + JS格式化卡号（每4位分隔））
115. 地址输入框（普通文本框 + 地址联想/省市区联动选择）
116. 多选项卡表单（分步骤表单，如注册流程：基本信息-验证信息-完成）
117. 内联表单（行内布局，如导航栏中的搜索表单）
118. 响应式表单（适配移动端/平板/桌面端，媒体查询调整布局）
119. 模态框表单（嵌入在弹窗中的表单，如新增/编辑弹窗）
120. 动态表单（可新增/删除表单条目，如多联系人信息录入）
121. 只读表单（所有控件设为readonly，用于展示已提交数据）
122. 离线表单（PWA技术，支持离线填写，联网后自动提交）
123. 跨域表单（带cors配置，提交数据到不同域名接口）
124. 文件预览表单（上传图片/视频后，原生JS/第三方插件实现预览）

## 五、小众/特殊场景表单类型
125. 自定义验证表单（通过pattern属性/JS自定义验证规则）
126. 无刷新提交表单（AJAX/Fetch提交，无需刷新页面）
127. 二进制数据表单（提交文件/图片二进制数据，type="file" + FormData）
128. 隐藏表单（CSS隐藏，用于后台自动提交数据，无需用户操作）
129. 只读不可复制表单（文本框 + oncopy/oncut事件禁止复制）
130. 自动填充表单（利用浏览器自动填充功能，如用户名/密码自动填充）
131. 无障碍表单（适配屏幕阅读器，含aria属性的表单控件）
132. 打印专用表单（CSS媒体查询适配打印样式，隐藏不必要控件）
